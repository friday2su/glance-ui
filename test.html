Notes

After installing the Glance UI extension:

  1. Visit this page
  2. Hold down the Alt key and click on any link
  3. The link should open in a pop-up modal instead of navigating away
  4. Use the X button to close the modal or click outside the modal to close it
  5. Use the arrow button to open the link in a new tab
